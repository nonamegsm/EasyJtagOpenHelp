EasyJTAG Classic software: 

Use "Scan" button for see visual pins connect as box connector pinout. Also info of connected pins is posted to log window as:

EasyJTAG Plus software (eMMC tool shuite):

Use "Scan pins" button next to the right of "Detect eMMC" button to see visual pin connection. EasyJTAG Plus software automatically close check pins windows after 3 second if all pins connected and go to eMMC detect process. 

 

Note: When used "Detect eMMC" button without "Detect pins" button and not all eMMC pins connected, eMMC tool shuite is try to use 4-bit or 1-bit connection mode (slowly as 8-bit) and post info about it to log window as: "Warning: BusWidth is automatically set to 1 Bit"

For use fastest 8bit mode user need set it at software again.
 

"EMMC File Manager" addon software as EasyJTAG Classic, post info about connected pins to log window.
