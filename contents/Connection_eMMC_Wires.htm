eMMC Wired connection

Wired eMMC connection is variety of ISP connection, but eMMC is not at PCB. It use for rarely eMMC types, for which there is no need to buy a socket, like BGA280 or BGA100 cases. At this connection can use 1bit mode for EasyJTAG 1st gen box or 1/4/8 bit modes for EasyJTAG Plus box.

 

For use 1/4bit modes connection with EasyJTAG Plus box the users can use ISP adapters (include with box).

 

For use 8bit mode connection with EasyJTAG Plus box the users needs to do hand-made PCB adapter and use it with e-Socket adapter.

 

At wired eMMC connection need use this pins:

  • GND (VSS, VSSQ)
  • VCC
  • VCCQ
  • DAT0 (1bit mode) or DAT0-DAT3 (4bit mode) or DAT0-7 (8bit mode)
  • CMD
  • CLK
  • VDDI

 

VDDI pin is an external capacitor pin. For this pin need solder capacitor 1uF to GND for Toshiba/Hynix eMMC. Samsung and Sandisk eMMC can work without it.

Many eMMC must be connect 2 GND's wires (VSS GND for VCC and VSSQ GNG for VCCQ).
