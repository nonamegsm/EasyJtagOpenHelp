EasyJTAG Plus Box Interface

Front view:

Rear view:

 

 

  1. Main 20pin interface. Use to connect JTAG adapter, SPI adapter, eMMC ISP adapter, eMMC Socket adapter, UFS socket adapter, etc
  2. RJ45 standart COM port bridge. Use as PC COM port interface (standart Unibox interface pinout)
  3. Power LED (lights up when  Box ready to work)
  4. Error LED (lights up when Box at Error state, flashes evenly when box at Firmware Upgrade mode )
  5. Recive Data LED (blinks during Recive Data )
  6. Transmit Data LED (blinks during Transmit Data)
  7. USB 2.0 Type B jack (interface for PC)
  8. Additional Power (5v) MicroUSB connector
  9. Smart Card (for software protection) connector

 

RJ45 standart COM port pinout (Unibox pinout):

  1. Vcc +5v
  2. TX
  3. RX
  4. CTS
  5. RTS
  6. MBUS
  7. RES
  8. GND

Additional MicroUSB power connector

Designed to reduce the load on the USB port of the PC when connecting a large load. Use to connect standart USB 5v 1-2A charger. 

When connected - it switch power for 20pin connector from PC to this port.

 

Smart Card connector

Use to connect Z3x protection smart card.
