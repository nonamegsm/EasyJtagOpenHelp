Introduse

The EasyJtag it's hardware and software complex which intended for the automated restoration of firmware of mobile phones, smartphones, tablets and other electronic devices, hereinafter referred to as devices using the JTAG interface and/or eMMC, UFS memory chips.

EasyJTAG was originally invented as JTAG Box, so no power was present on the connector. Further, when developing the eMMC functional connection, it was necessary to use external power regulators eMMC (2.8-3.3v core and 1.8-2.8-3.3v - IO (I/O interface levels)). Of course, it was possible to use one stabilizer at 2.8-3.3v, but this is appropriate only when the eMMC is connected to the adapter block. When connecting the ISP (In-System Programming) - soldering to the chip soldered to the board, you need to use 1.8 (IO) and 2.8 (core) in order not to damage the chips connected to these lines. Among other things, the support for SPI flash drives was added in order to support EasyJTAG/EasyJTAG Plus (external switched power supply regulator 1.8-2.8-3.3 is required for different chips according to the documentation).

 

Hardware 

The hardware is represented by two solutions. It's EasyJtag box and EasyJtagPlus box.

EasyJTAG box is first generation box which support:

  • JTAG connection (with native jtag 20pin cable)
  • ISP (eMMC at PCB) connection (with additional third party adapter)
  • eMMC e-Socket connection (1-bit mode with additional third party adapter)
  • SPI connection (with additional external chip power (1,8 or 3,3v) adapter)
 

EasyJTAG box

EasyJTAG Plus box is second generation box which support:

  • JTAG connection (with native Z3x adapter)
  • eMMC e-Socket connection (with native Z3x e-Socket adapter or additional third party e-Socket adapter)
  • ISP (eMMC at PCB) 1-bit mode connection (with native Z3x adapter)
  • SPI connection (with additional socket or soldering adapter)
  • NAND connection (with additional NAND Kit adapter)
  • M-PHY (UFS Memory) connection (with additional special UFS Memory socket adapter)

Software

EasyJTAG Plus is the Lightweight software with a friendly Interface, made especially for mobile phones repair, eMMC and UFS memory chips replacement and user data recovery.

The following types of connection to EasyJTAG:

  1. JTAG
  2. eMMC
  3. ISP eMMC
  4. HiPower CLK ISP eMMC
  5. SPI
  6. UFS

Features:

  • eMMC General Functionality: Read, Write, Erase, Change Boot Mode and HW Prtitioning Functions
  • Vendor Read Functions (Can read eMMC Data in Factory Firmware Pack for Samsung, Qualcomm, LG, MTK)
  • Vendor Write Function (Can directly flash Factory Firmwares in eMMC Chips Samsung, Qualcomm, LG, MTK, Huawei, Spreadtrum)
  • Android EXT4 file system Explorer
