Introduse

EasyJTAG Plus box  - Second generation box hardware, black-white label box, produced only at metal case, released and selling from December 2017, suppoterd eMMC, JTAG, ISP, SPI, NAND, M-PHY (UFS Memory) protocols.


Hardware

The hardware part of the complex is a device, hereinafter referred to as box.

 

General view

On the front of the box are four LEDs Blue, Red, Yellow and Green and two connectors 20-pin and RJ-45.

 

Front view

Glowing or flashing of LEDs indicate the status of the box.
The main connector is a 20-pin, two-line folder. The contacts of this connector are connected to the interface of the target device or its memory chip.
The RJ45 connector is auxiliary and is designed to enhance functionality of the box (USB-UART emulation, managed power lines, etc.).

On the back of the box are USB type-B and mini-USB ports and a slot for activation SIM card.

 

Back view

The USB type-B port provides power to the box and its connection to the computer running the Z3X EasyJtag software.
The mini-USB port allow connect additional power to the box in case if it needed.
A SIM card is inserted into the SIM slot, which is the hardware key to activate the box. Without the hardware key, EasyJtag box is impossible to operate.

Features

The main features of a new hardware are:

  • Updated Host MCU SAM3X8E vs LPC1768 in old box.
  • Updated HIGH SPEED FPGA SPARTAN6 vs SPARTAN3 in old box.
  • Updated POWER MANAGEMENT subsystem – Full Digital PWM with ADC control.
  • Added microUSB Connector if additional power required.
  • Added VCC and VCCQ switch with overcurrent protection.
  • Updated USB device interface: USB 2.0 480 Mbps (High Speed).
  • Integrated ISP engine: Z3X High Power Pro Engine (4MB/s).
  • Added eMMC 8 bit high speed interface up to 42Mhz.
  • Integrated NAND engine: NFC with 4 Kbyte RAM buffer and ECC.
  • Integrated M-PHY interface (UFS memory chip).
  • Integrated SPI engine: up to 21 MHz clock for popular SPI Memory Flash IC.
  • Integrated Self Test Core that exclude 90% faults in production.
  • Integrated CCID Card Reader for Z3X Software Use

Benefits

The main benefits of a new hardware are:

  • Box to Host PC speed increased up to 40MB/sec
  • JTAG Interface speed increased 20%
  • eMMC 1 Bit ISP Speed increased up to 4MB/sec
  • eMMC 4 Bit Speed up to 18MB/sec
  • eMMC 8 Bit Speed up to 28MB/sec ( eMMC 5.0+ )
  • Added NAND interface
  • Added M-PHY interface
  • Compatibility with 99 host PC systems and OS’s
  • Compatibility with old Easy JTAG gen. 1 ISP adapters (1bit)
  • Adapter-less ISP (all stuff inside the box)
  • Compatibility with all market popular eMMC sockets (with supplied adapters)
  • True Plug and Play for modern Windows OS (8, 8.1, 10)
  • Full Support for legacy OS like Windows XP
