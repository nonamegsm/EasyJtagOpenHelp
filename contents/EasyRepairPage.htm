Easy Repair Page

Easy Repair - work with devices through the JTAG interface in the "Easy Repair" mode - easy recovery, when the user selects the constructor file to automatically restore the boot area of the device.

 

 

In this mode, select the recovery file in the Repair File window, and then another Connection Manual appears next to the Log / JTAG Pinout / EMMC Pinout tabs - information on connecting the device in this particular case.

 

After that, we connect to the selected device according to the JTAG Pinout, and start the recovery Run Easy Repair.

 

Also on the first tab, the button Downloads files from support is added to download files from the support server, without using the standard shell z3x Shell.
