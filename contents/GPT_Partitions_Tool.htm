GPT Partitions Tool

This tab displays and allows you to work with flash partitions,
which are marked in the GPT table, and works with Dump Files (sector size 0x200).

 

The information is updated using the buttons on this panel:

eMMC - the name of the selected interface. Its color: green - ok, red - error (see log), blue - data update.

File - opens the selected dump file and allows you to work with it. Available only with dumps with a sector size of 0x200.

Reload - updates data by re-reading them from a previous source.

Cancel - cancels the task. Equivalent Stop Operation for any of the interfaces.

Read - reads selected sections.

Write - writes selected sections.

Erase - erases selected sections.

Fix Tables - fixes errors in GPT tables (main or backup), or recreates them in the absence. It is important that at least one is valid.

Create - creates a new partition (currently not work).

Delete - deletes the partition.

Resize - changes the partition size (works only for the last partition).

All - selects all partitions.

None -  removes selection from all partitions.

Additionally, when you right-click there is a context menu, which facilitates selection and allows you to read / read selected sections.
