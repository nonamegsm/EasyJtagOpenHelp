EasyJTAG box native SPI interface

EasyJTAG box SPI interface is simple and native. For use it need need create a simpe connector to SPI flash IC from 20pin EasyJTAG connector. Pull-up resistor (0.5-10k) need to switch WP (Write Protection) and HOLD pins to non-active state. Allso can be add 1-5uF capacitor for power line (VCC-GND) for best stability.

EasyJTAG 1st gen box  SPI interface need use external SPI chip power stabilizer from USB or RJ45 box connector 5v power. Voltage is SPI chips depended.
