Hardware Connection for eMMC

There are 4 types of eMMC connection:

 

  • Socket connection
  • ISP (In-System Programing) connection
  • Wired eMMC connection
  • Third-party PBC for soldering eMMC

 

Socket connection

EasyJTAG Plus box provide native eMMC socket with EasyJTAG Plus box selling. It's a high-quality tested interface for work with populat eMMC cases at hi-speed (8bit/42Mhz) and simple use.

EasyJTAG 1st gen box can use only 1bit eMMC access via third patry eMMC sockets. These sockets need additional power for eMMC via USB charger and may use third-party eMMC socket adapters.

 

ISP connection

 

ISP (In-System Programming) is not documented connection to eMMC at PCB. It use only 1-dat (1bit) mode to transder data between eMMC and box. 

There are 2 types of ISP eMMC connection:

  • Normal mode
  • Hi-Power CLK mode (tm)

 

Normal mode is use 6 wires for connection (CLK, CMD, Dat_0, GND, VCC, VCCQ lines)

Hi-Power CLK mode is use only 4 wires betweeen eMMC and box (CLK, CMD, Dat_0, GND lines) and power for PCB from USB charger

 

Wired eMMC connection

 

Wired eMMC connection is variety of ISP connection, but eMMC is not at PCB. It use for rarely eMMC types, for which there is no need to buy a socket, like BGA280 cases. Some eMMC brand  (Hynix, Toshiba) correct work only with capacitor at VDDi eMMC pin (1uF).

Third Party PCB for Soldering eMMC connection

Some manufacturers is created PCB for soldering as low-cost eMMC connecting solution. It may connect to EasyJTAG Plus box directly and for EasyJTAG 1st gen box with eMMC power additional adapter. This method can work at low frequency due to interference arising during operation at high frequencies due to simple PCB layout.
