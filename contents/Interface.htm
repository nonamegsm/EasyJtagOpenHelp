Software Interface

EasyJtag Classic User Software Interface contains tabs, checkboxes, buttons and other standard elements.

 

EasyJtag Classic User Interface

On the Left side of the main interface are:

  • Log Page
  • Jtag Pinout Helper
  • eMMC Pinout Helper
  • GPT Partitions Tool
  • Help System

On the Right side of the main interface are:

  • Easy Repair Page
  • JTAG Page
  • eMMC Page
  • USB Page
  • Settings Page
  • NAND Page (description in progress ...)

The Design and Functionality of the interface elements may change as they improve.
