Introduce

The EasyJtag it's hardware and software complex which intended for the automated restoration of firmware of mobile phones, smartphones, tablets and other electronic devices, hereinafter referred to as devices using the JTAG interface and/or eMMC, UFS memory chips.

 

EasyJTAG was originally invented as JTAG Box, so no power was present on the connector. Further, when developing the eMMC functional connection, it was necessary to use external power regulators eMMC (2.8-3.3v core and 1.8-2.8-3.3v - IO (I/O interface levels)). Of course, it was possible to use one stabilizer at 2.8-3.3v, but this is appropriate only when the eMMC is connected to the adapter block. When connecting the ISP (In-System Programming) - soldering to the chip soldered to the board, you need to use 1.8 (IO) and 2.8 (core) in order not to damage the chips connected to these lines. Among other things, the support for SPI flash drives was added in order to support EasyJTAG/EasyJTAG Plus (external switched power supply regulator 1.8-2.8-3.3 is required for different chips according to the documentation).
