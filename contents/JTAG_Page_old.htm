JTAG Page

JTAG Tool - is designed to work with the device through the JTAG interface. To do this, select the model of the device from the drop-down list at the top of the interface, and connect to the device according to the figure from the JTAG Pinout tab.

Typically, such a connection requires a connected battery and the device’s charging connector to USB, or the device should be connected after holding down the PowerOn key.

The Buttons:
Connect to Device - connection to the device;
Deatach Device - end of work with the device;
Stop Operation - to stop (cancel) selected operations with the device.

The 2 list menus:
Memory Chip - the memory section with which the work will take place;
JTAG Speed - speed of work on the JTAG interface.

The Buttons:
Find JTAG Pins - search for JTAG contacts for devices with missing connection information;
Reset Finder Config - installation of a standard JTAG connection diagram;
Load Config, Save Config - loading / saving the found JTAG pinouts;
Run Script - execution of sequences of commands (scripts).

Below are the 2 bookmarks:
Flasher - for working with the device’s memory through the JTAG interface;
Pinout - to reconfigure JTAG pinouts when searching for them.

The Flasher tab allows the user to:
1) read information about sections by clicking on the Read Partitions Data button;
2) read / erase / write the selected memory area of the device using the Start address selection window, a fixed length through the Preset or an arbitrary length using the Lenght;
3) select the modes of operation with the device’s memory: the inclusion of error correction ECC, the reaction on the Bad block.
