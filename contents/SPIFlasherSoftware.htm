EasyJTAG SPI Flasher

EasyJTAG SPI Flasher is native addon for EasyJTAG classic software. It have simple interface and big support chips database. EasyJTAG SPI Flasher is found at EasyJTAG Classic folder as EasyJtagSPI.exe

 

 

Interface:

  • EasyJTAG box connection
  • FT232 board interface
  • Bus Pirate SPI interface

 

Chip voltage:

  • 1,8v (1800mV)
  • 2,8v (2800mV)
  • 3,3v (3300mV)

 

Important: For select correct voltage for SPI chips use datasheet for its
