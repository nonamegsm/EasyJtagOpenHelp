Settings Page

Settings - this tab allows you to change program settings.

 

 

Firmware/Fullflash converter - on development stage;


eMMC Tool Options   - eMMC chip settings;


eMMC Bad Block Retry Count - setting the number of read retries of badly read or unreadable blocks in eMMC;


eMMC Bad Block Retry Delay - setting the timeout when reading bad or unreadable blocks in eMMC;


eMMC Erase Block Size - eMMC block size when erased;


Full eMMC Reinit after Bad Block Detected   - whether to do a full reset (initialization of connection) of eMMC upon detection of badly readable or unreadable blocks in eMMC;


Disable High Speed eMMC timing  - ban on the operation of the chip in the "High Speed Timing" mode (one of the extCSD eMMC registers is responsible for working in this mode;


Disable Write Optimisations - prohibition of optimization when writing data to eMMC;


Skip Block After Read Errors - skip unreadable blocks in eMMC (when reading dumps to extract residual information from eMMC);


Reinit box after exit software - re-initialize the hardware / box of the device to exit the program (for the correct launch of other programs working with the EasyJTAG / EasyJTAG Plus box);


Allow zero GPP Partitions Create - allow the creation of null GPP partitions;


Performing actions after the read/write task is finished: - allows you to select the action at the end of read / write operations: Do nothing / Play sound / Turn off the computer;


UFS Tool Options - Disable or Enable UFS safe mode;


Global Options - basic settings:

  • detection of custom SDcard Reader for plugin "SDcard Imaging Tool";
  • confirmation of the possibility of downloading photos from the program directory on bookmarks;
  • interface language selection;
  • move control panel to the left;
  • vertical partitions split.
