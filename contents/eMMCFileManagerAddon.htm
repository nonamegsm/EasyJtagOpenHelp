eMMC File Manager addon

eMMC File Manager (EFM) is third-arty addon for Z3x EasyJTAG box (1st and 2ns generation). 

 

EFM is designed for work with file-system based data at non-crypted partition (ext4/NTFS, etc). But now, when encryption of user data is widely used, the developer adds functions for flash firmware files to eMMC/UFS chips and others function similar to other Z3x software.

 

eMMC File Manager (EFM) releases is present for download from Z3x Shell. Beta software and support is available at Telegram group (http://t.me/z3xefm).

 

eMMC file manager main function:

  • Working with file systems EXT2, EXT3, EXT4, FAT12, FAT16, FAT32, exFAT, SELinux / XATTR support for EXT4
  • Reading a graphic key from the / data section or the "gesture.key" file for Android-Based platforms
  • Extraction of critical user information (Contacts, Call log, SMS) to VCF / TXT / HTML for Android-Based platforms
  • Read selected files and folders to specified location on hard drive
  • Skipping previously read files when re-saving to the selected folder
  • Bad sector read error handling
  • Reset user locks without data loss for Android-Based platforms
  • Working with the selected section (Read, Write, Erase, Format, Low Level Format)
  • Formatting the specified section, or the entire area in Ext2 / 3/4
  • Determining the type of encryption of the encrypted UserData section for Android-Based platforms using FDE, reading Mkey and Salt for further analysis and brute-force by third-party software (for advanced users)
  • Hex-view of the selected partition, or the entire source (EMMC / Dump / Disk)
  • Fast file browsing
  • Viewing File / Folder / SymLink Properties
  • Extended work with sections of EMMC memory. Reading EMMC Information (When using Easy Jtag / Easy Plus)
  • Work through Easy Jtag Box, Easy Jtag Plus Box, Mass Storage (USB Reader / KET / E-Mate / MOORС), QLM 9006, Dump (FullFlash) memory (full image or part of it).
  • Obtaining information about partitions from GPT, MBR, PMT, BCT, Low-level scanning (search for EXT4 file systems for a source with erased or unknown markup)
  • Hash and Salt Extraction to Crack Android Password (Crack Android PIN Code)
  • Viewing general information about a section
  • Reading device information for Android-Based platforms
  • Quick jump to the desired folder via Hot Link or the address bar
  • Unpacking / Packing Sparse Firmware Image

  • Add / Modify / Bulk Add / Modify SELinux / XATTR for Files / Folders (EXT4) *
  • Change / Bulk change UID / GID for selected files / folders (EXT4) *
